Welcome to Video View Controller.

This is a simple controller class to make using AVPlayerViewController easier.

Why?

I struggled with this for a while before finding the right solution. Mostly I pulled it together sing StackOverflow and docs and wanted to give back.

How to use this

You will need to create a AVPlayerViewController in Storyboard, connect it with a Segue (I use a modal segue for video display). And you'll need a valid NSURL. Pass both to the VideoDetailViewController constructor init(avVideoViewController:AVPlayerViewController, videoURL:NSURL). It'll just look after itself from there on in.
